# Tri-County Membership Manager

Website for Tri-County Gun Club built with Next.js 14 and Tailwind CSS.

[Live site](https://tri-county-membership-manager.onrender.com)

## Tech Stack

- Next.js 14 (App Router)
- JavaScript
- Tailwind CSS

## Setup

```bash
git clone https://github.com/Hawk-PDX/tri-county-membership-manager.git
cd tri-county-membership-manager
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Project Structure

Standard Next.js 14 app router setup. Pages are in `src/app/`, reusable components in `src/components/`.

## Deployment

Configured for Render with `render.yaml`. Can also deploy to Vercel, Netlify, etc.
